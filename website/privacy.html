1. 수집하는 개인정보

본 서비스는 회원가입을 받지 않으며, 이용자로부터 개인정보를 직접 수집하지 않습니다.

2. 자동 수집 정보

호스팅 제공자(Vercel)와 데이터베이스 제공자(Supabase)는 서비스 운영을 위해 다음 정보를 자동으로 수집할 수 있습니다:

  • 접속 IP 주소
  • 접속 시각, 브라우저 종류, 사용 OS
  • 이전 페이지 URL (referrer)

이 정보는 보안 모니터링 및 장애 대응 목적에 한해 사용됩니다.

3. 의료기관 정보 마스킹

본 서비스는 의료광고심의위원회 통과 시안의 광고 문구를 인덱싱하면서, 다음 정보를 자동으로 제거(마스킹)합니다:

  • 의료기관명 (병원/의원/클리닉/한의원/치과 등 고유명사)
  • 의료진 이름 (원장/교수/박사/전문의 등)
  • 전화번호, 팩스번호, 카카오톡 ID
  • 주소, 위치 안내 (역, 건물, 층 등)
  • 홈페이지 URL, SNS 계정

자동 마스킹이 100% 완벽하지 않을 수 있으며, 누락된 사례를 발견하시면 즉시 조치합니다.

4. 쿠키 사용

본 서비스 자체는 쿠키를 사용하지 않습니다. 다만 호스팅 제공자가 보안 목적으로 쿠키를 사용할 수 있습니다.

5. 외부 서비스

본 사이트는 다음 외부 서비스를 사용합니다:

  • Vercel (정적 호스팅)
  • Supabase (데이터베이스)
  • tailwindcss CDN, Chart.js CDN, jsdelivr CDN (스타일·차트)
