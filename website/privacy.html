개인정보처리방침

최종 개정일: 2026-04-26

1. 수집하는 개인정보

본 서비스는 회원가입을 받지 않으며, 이용자로부터 개인정보를 직접 수집하지 않습니다.

2. 자동 수집 정보

호스팅 제공자(Vercel)와 데이터베이스 제공자(Supabase)는 서비스 운영을 위해 다음 정보를 자동으로 수집할 수 있습니다:

  • 접속 IP 주소
  • 접속 시각, 브라우저 종류, 사용 OS
  • 이전 페이지 URL (referrer)

이 정보는 보안 모니터링 및 장애 대응 목적에 한해 사용됩니다.

3. 의료기관 정보 마스킹

본 서비스는 의료광고심의위원회 통과한 광고 문구를 인덱싱하면서, 다음 정보를 자동으로 제거(마스킹)합니다:

  • 의료기관명 (병원/의원/클리닉/한의원/치과 등 고유명사)
  • 의료진 이름 (원장/교수/박사/전문의 등)
  • 전화번호, 팩스번호, 카카오톡 ID
  • 주소, 위치 안내 (역, 건물, 층 등)
  • 홈페이지 URL, SNS 계정

자동 마스킹이 100% 완벽하지 않을 수 있으며, 누락된 사례를 발견하시면 즉시 조치합니다.

4. 웹사이트 분석 도구 사용

본 서비스는 사이트 이용 통계와 사용자 경험 개선을 위해 다음 분석 도구를 사용합니다:

  • Google Analytics 4 (Google LLC) — 방문자 수, 페이지뷰, 유입 경로, 사용 기기·브라우저, 접속 지역(국가/도시 단위) 통계 수집. IP 주소는 익명 처리되어 저장됩니다.
  • Microsoft Clarity (Microsoft Corp.) — 페이지 내 클릭 패턴(히트맵), 익명화된 세션 행동 분석. 입력 폼 내용은 자동 마스킹됩니다.
  • 네이버 웹사이트 애널리틱스(WCS) (네이버) — 네이버 검색 유입 분석.

분석 도구의 데이터 수집을 원하지 않으시면 브라우저의 "Do Not Track" 옵션 또는 광고 차단 확장 프로그램을 통해 차단할 수 있습니다.

5. 쿠키 사용

본 서비스는 위 분석 도구의 정상 동작을 위해 일부 쿠키를 사용합니다(방문 식별, 세션 추적). 호스팅 제공자(Vercel)도 보안 목적으로 쿠키를 사용할 수 있습니다.

6. 외부 서비스

본 사이트는 다음 외부 서비스를 사용합니다:

  • Vercel (정적 호스팅)
  • Supabase (데이터베이스)
  • Google Analytics 4 / Microsoft Clarity / 네이버 WCS (분석 — 4항 참조)
  • tailwindcss CDN, Chart.js CDN, jsdelivr CDN, Pretendard CDN (스타일·폰트·차트)

7. 문의

개인정보 관련 문의는 사이트 내 검색 결과의 [오류 제보] 버튼 또는 문의 페이지를 이용해주세요. 관리자에게 자동 전달됩니다.
